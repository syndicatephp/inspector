# Model Inspections for Laravel // Filament

## Installation

You can install the package via composer:

```bash
composer require syndicatephp/inspector
```

## Usage

```php
$inspector = new Syndicate\Inspector();
echo $inspector->echoPhrase('Hello, Syndicate!');
```

## Testing

```bash
composer test
```
